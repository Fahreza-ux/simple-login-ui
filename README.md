# # Simple Login UI Boilerplate

Proyek ini adalah template halaman login sederhana yang dibangun menggunakan **HTML5** dan **CSS3**. Dibuat sebagai bahan pembelajaran pengembangan antarmuka web (Frontend Development) untuk memahami struktur form dan *styling* dasar.

## 🚀 Fitur
- **Responsive Design**: Tampilan menyesuaikan ukuran layar (Desktop & Mobile).
- **Clean Code**: Struktur kode yang rapi dan mudah dipelajari.
- **Modern UI**: Menggunakan teknik Flexbox untuk tata letak yang presisi.

## 🛠️ Teknologi yang Digunakan
- **HTML5**: Untuk struktur semantik.
- **CSS3**: Untuk desain dan tata letak (Flexbox).
- **Google Fonts**: (Opsional) Jika ingin menggunakan font kustom.
